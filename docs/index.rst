============================
UK NSC Project Documentation
============================

..
    To build and view this documentation in a browser, set up a Python 3.6+ virtual
    environment then run from the project root::

      pip install -r requirements-dev.txt
      sphinx-build docs docs/_build
      python -mwebbrowser docs/_build/index.html
